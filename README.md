### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/igaimerca/frontend-challenge.git
   ```
2. Install NPM packages
   ```sh
   cd fronted-challenge
   npm install
   ```
2. Run in DEV mode
   ```sh
   npm start
   ```
